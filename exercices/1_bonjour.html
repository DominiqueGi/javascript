Bonjour

  • Créer un fichier bonjour.htmml et faites le pointer vers un fichier bonjour.js
  • Ecrire le code invitant l'utilisateur à saisir d'abord son nom et ensuite son prénom dans deux boîtes de dialogue successives.
  • Afficher ensuite dans une autre boîte de dialogue le nom et le prénom de l'utilisateur.
  • Attention à bien respecter les espaces entre les mots dans le message final.
